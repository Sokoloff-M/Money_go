# Учет движения денежных средств (ДДС)

Веб-приложение для учета финансовых операций с поддержкой справочников и фильтрации.

## Возможности

- 📝 CRUD операции для транзакций
- 📊 Фильтрация по дате, статусу, типу, категории
- 📚 Управление справочниками (статусы, типы, категории, подкатегории)
- 🔗 Логические зависимости между справочниками
- ✅ Валидация на клиенте и сервере

## Технологии

- Python 3.11+
- Django 5.0
- SQLite
- Bootstrap 5
- jQuery
